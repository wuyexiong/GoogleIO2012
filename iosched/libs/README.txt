The following libraries are required to successfully build:

android-support-v4.jar
  - The Android Support Library (r8 or later)
    Install from the Android SDK Manager.
gcm.jar
  - The Google Cloud Messaging for Android Library
    Install from the Android SDK Manager.
google-api-plus-v1-rev4-java-1.5.0-beta.jar
  - The Google+ API Client Library for Java
    See http://code.google.com/p/google-api-java-client/wiki/APIs#Google+_API

google-api-client-1.8.0-beta.jargoogle-api-client-android2-1.8.0-beta.jar
  - The Google APIs Client Library for Java
    See http://code.google.com/p/google-api-java-client/
google-http-client-1.8.3-beta.jargoogle-http-client-android2-1.8.3-beta.jar
  - The Google HTTP Client Library for Java
    See http://code.google.com/p/google-http-java-client/
google-oauth-client-1.8.0-beta.jar
  - The Google OAuth Client Library for Java
    See http://code.google.com/p/google-oauth-java-client/
gson-2.1.jar
  - Google Gson
    See http://code.google.com/p/google-gson/
guava-11.0.1.jar
  - Guava
    See http://code.google.com/p/guava-libraries/
jackson-core-asl-1.9.4.jar
  - Jackson
    See http://jackson.codehaus.org/

protobuf-java-2.2.0.jar
  - Protocol Buffers
    http://code.google.com/p/protobuf/
